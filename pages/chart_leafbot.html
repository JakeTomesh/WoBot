Chart Leafbot

How to use:

(click for instructions)

1. Use command "=charter" in Command channel in Dawn Endgame Bids server

2. Click & drag, highlight all bids to be charted from Leafbot

3. Copy & paste bids to text area

4. Click chart button for VKP or Non-VKP bids

5. Use "Copy next/previous bids" button to enter into bot

6. Input all Non-VKP bids into Jellobot via "closed-bids" channel in Dawn Endgame Bids server

7. Input all VKP bids into Chatpate via "chatpate-input" channel in Dawn server
